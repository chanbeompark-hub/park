# FitBlog Pro - 피트니스 전문 블로그 마케팅 사이트

피트니스 업계에 특화된 블로그 마케팅 서비스를 제공하는 전문 업체의 웹사이트입니다.

## 🏋️ 프로젝트 개요

**FitBlog Pro**는 피트니스 센터, 요가 스튜디오, 개인 트레이너, 영양 상담사 등 피트니스 업계 종사자들을 위한 전문 블로그 마케팅 서비스를 제공합니다. 브랜드 인지도 향상부터 고객 유치까지 종합적인 디지털 마케팅 솔루션을 제공합니다.

## 📋 페이지 구성

### 1. 헤더 (Header)
- 고정형 네비게이션 바
- 브랜드 로고 (덤벨 아이콘)
- 주요 섹션으로의 스무스 스크롤 링크
- 반응형 모바일 메뉴 (햄버거 메뉴)

### 2. 히어로 섹션 (Hero Section)
- 강력한 메인 메시지와 CTA
- 핵심 성과 지표 (500+ 프로젝트, 150% 트래픽 증가, 98% 고객 만족도)
- 애니메이션 카운터 효과
- 고품질 피트니스 이미지

### 3. 서비스 섹션 (Services)
- **전문 콘텐츠 제작**: 운동 가이드, 영양 정보, 피트니스 트렌드
- **SEO 최적화**: 키워드 리서치, 온페이지 SEO, 백링크 구축
- **소셜미디어 관리**: 인스타그램, 유튜브, 페이스북 통합 마케팅
- **데이터 분석**: 트래픽 분석, 전환율 측정, 월간 성과 리포트

### 4. 포트폴리오 섹션 (Portfolio)
- 실제 성공 사례 3개 소개
  - 프리미엄 헬스장: 온라인 예약 300% 증가
  - 요가 스튜디오: 브랜드 인지도 대폭 상승
  - 개인 트레이너: 개인 브랜딩을 통한 고객 확보
- 구체적인 성과 수치 표시

### 5. 고객 후기 섹션 (Testimonials)
- 실제 고객들의 생생한 후기
- 고객 프로필 이미지와 직업 정보
- 신뢰도를 높이는 구체적인 경험담

### 6. 요금제 섹션 (Pricing)
- **스타터**: 월 30만원 (기본 서비스)
- **프로페셔널**: 월 60만원 (인기 플랜, 추천)
- **엔터프라이즈**: 월 120만원 (프리미엄 서비스)
- 각 플랜별 상세 기능 비교

### 7. 상담 신청 섹션 (Contact)
- 상세한 문의 폼
  - 개인 정보 (이름, 업체명, 연락처, 이메일)
  - 사업 유형 선택 (헬스장, 요가스튜디오, 개인트레이너 등)
  - 예상 예산 선택
  - 자유 문의 내용
- 폼 유효성 검사 및 실시간 피드백
- 개인정보 동의 체크박스

### 8. 푸터 (Footer)
- 회사 정보 및 브랜드 소개
- 서비스 링크
- 회사 정보 링크
- 연락처 정보
- 소셜미디어 링크

## ⚡ 기능 구현

### JavaScript 기능
1. **스크롤 애니메이션**: Intersection Observer API 활용
2. **숫자 카운터 애니메이션**: 히어로 섹션 통계 수치
3. **스무스 스크롤**: 네비게이션 클릭 시 부드러운 이동
4. **폼 유효성 검사**: 실시간 입력 검증
5. **알림 시스템**: 성공/오류 메시지 표시
6. **모바일 메뉴**: 반응형 햄버거 메뉴
7. **이미지 지연 로딩**: 성능 최적화

### CSS 기능
1. **CSS Grid & Flexbox**: 현대적 레이아웃
2. **CSS 변수**: 일관된 디자인 시스템
3. **반응형 디자인**: 모바일 퍼스트 접근
4. **호버 효과**: 인터랙티브한 사용자 경험
5. **애니메이션**: 부드러운 전환 효과
